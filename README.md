# Magister Auto-Login

Automagically logs into Magister 6 so you won't have to.

## How to install

Click [here](https://mb-o.nl/autologin) and click on "Add to Browser".

## Built With

* [Bootstrap](https://getbootstrap.com) - CSS framework used
